// Program to perform Deep Copy for String 

public class DeepCopyString {
    public static void main(String[] args) {
        // Original string
        String original = "Hello Java";

        // Deep copy using new String()
        String copy = new String(original);

        // Display both strings
        System.out.println("Original String: " + original);
        System.out.println("Copied String: " + copy);

        // Check if both references are same
        if (original == copy) {
            System.out.println("Both strings refer to the same object (shallow copy).");
        } else {
            System.out.println("Both strings are different objects (deep copy).");
        }

        // Check if contents are equal
        if (original.equals(copy)) {
            System.out.println("Contents of both strings are equal.");
        }
    }
}

OUTPUT

 Original String: Hello Java
Copied String: Hello Java
Both strings are different objects (deep copy).
Contents of both strings are equal.
