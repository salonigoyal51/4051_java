// Program to prove String is immutable programmatically 

    public class StringImmutableDemo {
    public static void main(String[] args) {
        String original = "Hello";

        // Try to modify the string
        String modified = original.concat(" World");

        // Display the strings
        System.out.println("Original String: " + original);
        System.out.println("Modified String: " + modified);

        // Check if both references are same
        if (original == modified) {
            System.out.println("Both refer to the same object.");
        } else {
            System.out.println("Both refer to different objects.");
        }
    }
}

OUTPUT

   Original String: Hello
Modified String: Hello World
Both refer to different objects.
