//Program to insert the string using StringBuffer class 

public class StringBufferInsert {
    public static void main(String[] args) {
        // Create a StringBuffer object
        StringBuffer sb = new StringBuffer("Hello World");

        // Insert a string at index 6
        sb.insert(6, "Java ");

        // Display the result
        System.out.println("After insertion: " + sb);

        // Insert another string at the beginning
        sb.insert(0, "Welcome! ");

        // Display the updated StringBuffer
        System.out.println("After inserting at the beginning: " + sb);
    }
}

OUTPUT

    
After insertion: Hello Java World
After inserting at the beginning: Welcome! Hello Java World
