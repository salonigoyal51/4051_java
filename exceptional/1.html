//program that throws an exception and catch it using a try-catch block. 

        
   public class ExceptionHandlingExample {

    public static void main(String[] args) {
        try {
            System.out.println("Inside try block");

            // Deliberately throw an exception
            throw new ArithmeticException("This is a custom arithmetic exception");

            // Any code here will not execute
            // System.out.println("This line will not execute");
        } 
        catch (ArithmeticException e) {
            System.out.println("Exception caught!");
            System.out.println("Exception message: " + e.getMessage());
        }

        System.out.println("Program continues after try-catch block");
    }
}

Exception caught!
Exception message: This is a custom arithmetic exception
Program continues after try-catch block
