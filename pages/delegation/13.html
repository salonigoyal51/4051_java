//Program to Close the Frame using WindowAdapter Class 

import java.awt.*;
import java.awt.event.*;

public class CloseFrameExample extends Frame {

    public CloseFrameExample() {
        // Frame title
        super("Close Frame Example");

        // Layout
        setLayout(new FlowLayout());

        // Add a label
        Label label = new Label("Click the close button (X) to close the frame.");
        add(label);

        // Frame properties
        setSize(400, 150);
        setVisible(true);

        // Add WindowAdapter to handle window closing
        addWindowListener(new WindowAdapter() {
            public void windowClosing(WindowEvent e) {
                System.out.println("Frame is closing...");
                dispose(); // Close the frame
            }
        });
    }

    public static void main(String[] args) {
        new CloseFrameExample();
    }
}

OUTPUT

Initial Frame:

Click the close button (X) to close the frame.


After clicking X:

Frame is closing...


The frame closes, and the program terminates (if no other frames are open).
